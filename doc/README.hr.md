## Kako instalirati CinemaPress?

> #### Instalirajte putem web-lokacije

[https://cinemapress.sh](https://cinemapress.sh)

[![Instalirajte putem web-lokacije](https://raw.githubusercontent.com/CinemaPress/CinemaPress/master/themes/default/public/admin/images/min/web.png)](https://cinemapress.sh)

> #### Instalirajte putem terminala

`bash <(wget cinemapress.sh -qO-)`

![Instalirajte putem terminala](https://raw.githubusercontent.com/CinemaPress/CinemaPress/master/themes/default/public/admin/images/min/cli.png)
